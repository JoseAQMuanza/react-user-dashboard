# React User Dashboard

This is a user management application built with **ReactJS**, **React Router**, and **TailwindCSS**. It includes features like:

- User login (mocked)
- Protected routes
- Add/Edit/Delete users
- Form validation
- Dynamic routing
- Clean responsive UI with TailwindCSS
